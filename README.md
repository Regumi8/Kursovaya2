# МЕЧ И ТЬМА

Минималистичная текстовая ролевая игра, работающая в терминале. Проект построен на модульной архитектуре JavaScript (ES Modules) с использованием встроенных возможностей Node.js.

## О проекте

Это текстовая игра, разработанная на Node.js, которая позволяет игроку погрузиться в мир приключений, взаимодействуя с персонажами, собирая предметы и сражаясь с противниками. Игра использует JSON для хранения диалогов и настроек, а также файловую систему для сохранения прогресса игрока.

## 🚀 Особенности игры

•   **Динамичные диалоги:** Игра основана на древовидной структуре диалогов, позволяющей игроку выбирать варианты ответов и влиять на ход беседы.
•   **Боевая механика:** Пошаговые сражения с расчетом урона, критических ударов, защиты противников и системой исцеления.
•   **Прокачка и экипировка:** Улучшение характеристик через покупку или нахождение предметов (оружие, броня, артефакты).
•   **Умное суммирование:** Система контроля бонусов (эффекты брони не складываются бесконечно, а выбирается лучший вариант).
•   **Сохранение прогресса:** Полноценная система Save/Load, использующая файловую систему (`fs`). Данные игрока и текущий момент истории сохраняются в JSON.
•   **Управление через консоль:** Интуитивно понятное управление с помощью ввода цифр или команд (exit, s, d, i, 0).
•   **Модульная структура:** Проект разбит на модули (player, handlers, dialogueTree, saveManager, constants), что облегчает разработку и поддержку.

## 🛠 Технологический стек

•   **Node.js** (v18+)
•   **JavaScript (ESM)**
•   **FileSystem (fs)** - для работы с сохранениями.
•   **Readline** - для интерактивного ввода в консоли.

## 📂 Структура проекта

•   `js.js` - точка входа, основной игровой цикл и обработка команд.
•   `player.js` - логика персонажа, расчет характеристик, инвентарь и статус.
•   `saveManager.js` - модуль управления файлами сохранений (создание, загрузка, удаление).
•   `handlers.js` - обработчики игровых событий (битвы, магазины, специальные действия).
•   `dialogueTree.json` - база данных всех игровых сцен и переходов.
•   `bat.js` - вспомогательные функции для расчетов (криты, лимиты HP).

## 🎮 Управление

Во время диалога вы можете вводить номера вариантов ответа или специальные команды:

•   `0` - Посмотреть статус персонажа (HP, текущий урон, надетые вещи).
•   `i` - Открыть инвентарь.
•   `save` - Быстрое сохранение прогресса.
•   `delete` — Удаление текущего сохранения (с подтверждением).
•   `exit` - Сохранить игру и выйти в консоль.
•   `help` - Посмотреть все команды прямо в игре.

## 📦 Установка и запуск

1. Убедитесь, что у вас установлен **Node.js**.
2. Скачайте файлы проекта в одну папку.
3. Откройте терминал в этой папке.
4. Запустите игру командой: node js.js

## 💾 Система сохранений

Игра автоматически создает папку /saves/ в корне проекта. 

•   `player_save.json` - хранит параметры персонажа и инвентарь.
•   `dialogue_save.json` - хранит ключ последнего узла диалога, на котором вы остановились.
